# App Aterramento Subestação NBR 15751

Aplicação web para cálculo de parâmetros de aterramento de subestações conforme NBR 15751.

## Funcionalidades
- Cálculo de Tensão de Passo Máxima Admissível
- Cálculo de Tensão de Toque Máxima Admissível
- Interface intuitiva com Tailwind CSS

## Como Rodar

1. Certifique-se de ter o Node.js instalado.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Abra o navegador no endereço indicado (geralmente http://localhost:5173).

## Tecnologias
- React
- TypeScript
- Vite
- Tailwind CSS
